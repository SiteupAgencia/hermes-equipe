# Hermes — instalação da equipe

Bem-vindo! Em 4 passos você deixa o Hermes pronto. Você só conversa; a IA faz o resto.

**Antes de começar:** você precisa ter o **Claude Code (app desktop)** instalado.

1. Baixe esta pasta: botão verde **Code → Download ZIP** aqui em cima, e descompacte (ex.: na Área de Trabalho).
2. Abra o **Claude Code desktop**.
3. No app, escolha **Select folder** e selecione a pasta `hermes-equipe` que você descompactou.
4. Na conversa, escreva: **instalar o hermes**

Pronto. A IA vai te pedir seu e-mail e senha do Hermes, instalar o que falta e te dar as boas-vindas.

Dúvida? Fale com o Gabriel ou o Lucas.
